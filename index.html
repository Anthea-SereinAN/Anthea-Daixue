带学背诵管理系统

Daixue Recitation Manager

基于艾宾浩斯遗忘曲线，为老师和学生量身打造的背诵进度管理工具。多学生、多科目、自动计算复习节点——所有数据存储在本地浏览器，无需注册即可开始使用。

🚀 立即体验

✦ 核心功能

🧠
艾宾浩斯自动排期
录入背诵条目后，系统自动按遗忘曲线计算复习日期：当天 → 1 → 2 → 4 → 7 → 15 → 30 → 45 → 60 → 75 → 90 天，共 11 个复习节点
📊
7 种状态追踪
每个复习格子有独立状态：按时完成 / 提前完成 / 逾期补完 / 今日待复习 / 逾期未复习 / 未来待复习 / 已跳过，一目了然
👥
多学生 × 多科目
支持为多位学生分别建立科目，独立管理背诵进度。适合带学老师、班主任、考研辅导等场景
📅
日计划 + 周计划
日计划按上午/下午/晚上三时段安排，周计划按专业课/英语/政治三科分类。支持结转、复制、批量导入
📥
导入 / 导出
数据可导出为 JSON（备份恢复）和 Markdown（可读打印）。导入前自动校验、预览数据量、备份当前数据
🌙
暗色模式
内置暗色/亮色切换，背诵表格在两种模式下均有清晰可读的色彩编码
